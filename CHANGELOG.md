# Changelog

Форк [pengvench/MTProxyAutoSwitch](https://github.com/pengvench/MTProxyAutoSwitch).

## 1.6 — 2026-06-19

### Исправлено

- **Telegram-источники:** галка «Использовать Telegram-источники» сохраняется между запусками и больше не сбрасывается при проверке авторизации.
- **Трей:** при сворачивании и восстановлении из трея открывается главная страница, а не последний раздел настроек.
- **Списки proxy:** рабочие списки читаются и сохраняются в `%APPDATA%\MTProxyAutoSwitch\list\`, а не только в папке установки (важно после переустановки/обновления).
- **Обновление пула:** если strict-проверка (Deep media / Whitelist) отклонила все новые proxy, прежний рабочий пул и `proxy_list.txt` не затираются.
- **Загрузка пула:** при старте восстанавливается полный сохранённый пул из `proxy_list.txt` / `report.json`, а не урезанный `fast_list.txt` (24 proxy).
- **Закрытие:** при выходе текущий рабочий пул сохраняется на диск.
